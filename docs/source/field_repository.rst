6. Repository (R)
====================

``datacite:publisher``

A repository is a storage location from which the research product can be retrieved.

**Usage Instruction**

This element contains the name of the research product repository.

**Example**

.. code-block:: xml
   :linenos:

   <publisher>
     BioVel
   </publisher>


.. code-block:: xml   
   :linenos:
  
  <publisher>
     protocols.io
   </publisher>
